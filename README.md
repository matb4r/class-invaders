# Class Invaders game
Simple game based on [Space Invaders](https://en.wikipedia.org/wiki/Space_Invaders).

[![](http://img.youtube.com/vi/fz2shNlDVoc/0.jpg)](https://www.youtube.com/watch?v=fz2shNlDVoc)

### Keyboard:

| Key        | Effect           |
| ------------- |-------------|
|LEFT|Move ship left|
|RIGHT|Move ship right|
|SPACE / D|Shoot|
|ESCAPE|Close game|
